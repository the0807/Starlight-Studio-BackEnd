# 🖥️ 주요 기능
- 제목, 주제, 캐릭터, 배경을 입력하면 ChatGPT API를 통해 동화 내용 생성
- 생성된 내용 저장
- 내용 재생성
- 요청사항을 작성하고 반영된 내용 재생성
- 동화 내용에 따른 이미지 생성
- 동화 미리보기
- 동화 공유하기
